# Managed Form Defaults

Extraccion source-derived del runtime original de Rental para defaults de formulario, serie/sucursal y previews del documento.

## Estado

- `source-derived`
- conserva rutas, labels y campos del flujo original de Rental

## Uso recomendado

- conservarlo solo para trazabilidad o adapters legacy
- usar paquetes canonicos para integraciones nuevas

## Reemplazos canonicos

- `form-defaults-surface`
- `form-preview-surface`
- `form-header-identity-surface`
